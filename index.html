Casio fx-97SG X

Calculator Guide for Math

Based on the official Casio fx-97SG X User's Guide
1. Standard Form
Displaying results in standard form (Sci mode)
Steps
  1. Press SHIFT + MENU/SETUP to open the setup menu
  2. Select Number Format
  3. Select Sci, then choose number of significant figures (e.g. 3)
  4. All results now display as e.g. 3.20×10⁵
To return to normal display: [SHIFT] + [MENU/SETUP] → Number Format → Norm → select Norm 2 (recommended).
Norm 1 vs Norm 2 & ENG
Norm 1: Switches to standard form for numbers ≥ 10¹⁰ or ≤ 10⁻²
Norm 2: Only switches for numbers ≥ 10¹⁰ or ≤ 10⁻⁹ — shows more numbers as plain decimals. Recommended for most use.
ENG key: Shifts the decimal point in steps of 3 (kilo, mega, milli etc.). Useful for physics.
2. Mean & Standard Deviation

The fx-97SG X uses Statistics Mode for calculating mean and standard deviation. The Frequency column is hidden by default — turn it on in Setup first if your data has repeated values.

Enable the Frequency column
Setup
  1. Press SHIFT + MENU/SETUP
  2. Scroll down and select Statistics
  3. Select On to show the Freq column
If all your data values appear only once, leave Statistics set to Off and skip the Freq column entirely.
Entering Statistics Mode
Steps
  1. Press MENU → highlight the Statistics icon → press =
  2. On the Select Type screen, press 1 to select 1-Variable
  3. The Statistics Editor appears — this is where you enter your data
Inputting your data
Steps
  1. Type your first data value in the x column, press =
  2. If Freq is ON: move to the Freq column and enter how many times that value appears, press =
  3. Repeat for all values, pressing = after each entry
  4. When done, press AC — data is saved and you move to the calculation screen
To edit or insert a row:
Press OPTN → Data — from here you can edit values and insert new rows.

Alternatively, to delete a single row: move the cursor to that row and press DEL.
Getting your results
Steps
  1. Press OPTN
  2. Select 1-Variable Calc (usually option 3)
  3. A list of results appears — scroll with the arrow keys
x̅: The arithmetic mean
σx: Population standard deviation
Σx: Sum of all values
Σx²: Sum of squares
To return to normal calculator mode: press MENU → select Calculate (icon 1).
3. Time Conversion

The o' '' key (3rd row, 2nd from left — next to the (-) key) handles both degrees/minutes/seconds and hours/minutes/seconds, since both use base-60.

Decimal → hours, minutes, seconds
Steps
  1. Type the decimal value — e.g. 2.575
  2. Press =
  3. Press o' '' — result displays as 2°34'30" meaning 2h 34min 30sec
2.575 hours → 2h 34min 30sec
Hours, minutes, seconds → decimal
Steps
  1. Type the hours value, then press o' ''
  2. Type the minutes value, then press o' ''
  3. Type the seconds value, then press o' ''
  4. Press SHIFT + = — result shown as a decimal
2+ o' ''+ 34+ o' ''+ 30+ o' ''+ SHIFT+= →  2.575
You must enter something for every unit — even if it is zero.

Example: 2 hours and 32 seconds (no minutes):

2 + o' '' + 0 + o' '' + 32 + o' '' + SHIFT+=

The minutes must be entered as 0 — you cannot skip it.
4. Simultaneous Equations
Entering Equation Mode
Steps
  1. Press MENU → highlight the Equation icon → press =
  2. Press 1 to select Simul Equation
  3. Press 2 for 2 unknowns (x and y), or 3 for 3 unknowns
Rearrange equations to the form ax + by = c before entering. All terms must be on the left side.
e.g.  3y = 8 − 2x  →  2x + 3y = 8
Entering coefficients — format: a₁x + b₁y = c₁
Steps
  1. Enter a₁ (coefficient of x in equation 1) → press =
  2. Enter b₁ (coefficient of y in equation 1) → press =
  3. Enter c₁ (the constant on the right side) → press =
  4. Repeat for equation 2: enter a₂, b₂, c₂ pressing = after each
  5. Press = to display x — press = again to display y
2x + 3y = 8 and x − y = 1 Enter: a₁=2, b₁=3, c₁=8, a₂=1, b₂=−1, c₂=1 Result: x=−1, y=2
Press AC at any time to clear all coefficients and start again.
5. Solving Quadratic & Cubic Equations
Entering Equation Mode
Steps
  1. Press MENU → highlight Equation → press =
  2. Press 2 to select Polynomial
  3. Press 2 for quadratic (degree 2) or 3 for cubic (degree 3)
Before entering coefficients: The equation must equal 0. Move all terms to one side first.
e.g.  x² = 5x − 6  →  x² − 5x + 6 = 0
Quadratic: ax² + bx + c = 0
Steps
  1. Enter a → press =
  2. Enter b → press =
  3. Enter c → press =
  4. Press = to see x₁ — press again for x₂
  5. Press = once more on the last solution to return to the Coefficient Editor
x² − 5x + 6 = 0 → a=1, b=−5, c=6 → x₁=3, x₂=2
Complex/imaginary roots are shown too (e.g. −1+i, −1−i). This happens when the discriminant b²−4ac is negative.
Cubic: ax³ + bx² + cx + d = 0
Steps
  1. Enter a, b, c, d one by one, pressing = after each
  2. Press = to cycle through x₁, x₂, x₃
x³ + 0x² − 4x − 4 = 0 → a=1, b=0, c=−4, d=−4 → x=−1, −2, 2
If a term is missing (e.g. no x² term), you must still enter 0 for that coefficient. Do not skip it.
6. Trigo Exact Form Using the Calculator

Your calculator gives decimal answers for sin, cos and tan. But for special angles, you can work backwards to find the exact form (surds/fractions) using this squaring trick.

The Method
Steps
  1. Key in your trigo value — e.g. sin 45
    You get a decimal like 0.70710...
  2. Square the result — you get a clean fraction e.g. 1/2
  3. The exact form is the square root of that fraction — with the sign from the original decimal
Example 1 — sin 60°
  1. sin 60° = 0.86602...  (positive)
  2. Square the result → you get 3/4
  3. Original was positive → sin 60° = √3√4
sin 60° = √3√4 = √32
Example 2 — cos 150°
  1. cos 150° = −0.86602...  (negative)
  2. Square the result → you get 3/4
  3. Original was negative → cos 150° = −√3√4
cos 150° = − √3√4 = − √32
Example 3 — tan 60°
  1. tan 60° = 1.73205...  (positive)
  2. Square the result → you get 3  (whole number — no fraction)
  3. Original was positive → tan 60° = √3
tan 60° = √3
